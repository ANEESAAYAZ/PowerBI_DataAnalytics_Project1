# PowerBI_DataAnalytics_Project1

-	Perform all the necessary data cleaning tasks that you think are necessary to all the datasets shared.

-	Other than the basic cleaning operations make sure to follow the tasks for the datasets as listed:
1.	FlightData.xlsx – Adjust the datatypes to the most appropriate options. Also we have a column named “Flight Duration” which contains the duration of the entire flight. Split the column to two – One for showing the Hours and the other for the minutes and remove the original data column. Specially note to delete the “hr” and the “min” suffixes from the column

2.Employee_layoffs.xlsx – The dataset contains the informations of Employee Layoffs for various companies over the past years. Make sure all the necessary cleaning actions are done and the datatypes are adjusted accordingly.

3.Customer_Data.xlsx – The data set contains information related to some of the customers. Perform all the data cleaning steps required. Along with the same we have a column named “contact info” which holds both the email and phone number of the customer. Split the details into two separate columns and also adjust the datatypes.

4. Amazingcart.xlsx – Perform all the basic data preprocessing steps. Ensure that there are no missing and duplicate values and adjust the datatypes effectively.
